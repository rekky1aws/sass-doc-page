Introduction à SASS.

Installation

Installation de la version Ruby de SASS.

$ sudo apt install ruby-sass

Pour des raisons de practicité, nous allons aussi installer le mode watch

$ sudo apt install ruby-listen
$ sudo gem install sass-listen

Utilisation

Pour utiliser SASS, il va falloir un fichier source en .scss, la syntaxe classique de SASS. Cette syntaxe contient le CSS, cela veut dire que tout code CSS est valide en SCSS. On va ensuite pourvoir compiler le fichier grace a cette commande.

$ sass < src_file >:< destination_file >

Grace au reste de l'installation que nous avons fait ci-dessus, nous allons pouvoir utiliser le paramètre --watch pour faire en sorte que le fichier se compile automatiquement a chaque changement sur le fichier source.

$ sass < src_file >:< destination_file > --watch
